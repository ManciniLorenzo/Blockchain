
------------------Blockchain in PHP------------------

Posizionare la cartella del progetto in 
X:\xampp\htdocs\Blockchain

Per accedere al sito digitare nella barra di ricerca:
localhost/Blockchain/public

Se non si possiede un account occorre registrarsi.

Una volta registrati, accedere con le proprie credenziali.
